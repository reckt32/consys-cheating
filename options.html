Gemini Screenshot Helper Options

You need a Gemini API key to use this extension. Get your key from Google AI Studio.

Configure the keyboard shortcut in Brave's extension settings:

  1. Navigate to brave://extensions/shortcuts in your address bar.
  2. Find "Gemini Screenshot Helper" in the list.
  3. Click the edit (pencil) icon next to the command and press your desired key combination (e.g., Ctrl+Shift+G or Command+Shift+G).
